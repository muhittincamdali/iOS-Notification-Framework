# Changelog

All notable changes to NotificationKit will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-04

### Added
- Initial release
- Type-safe notification builder pattern
- Support for local notifications
- Time interval triggers
- Calendar-based triggers
- Location-based triggers (CoreLocation)
- Interactive notification categories
- Notification actions (tap, text input, destructive)
- iOS 15+ Time Sensitive support
- iOS 15+ Critical alerts support
- Thread-based notification grouping
- Relevance score for notification ordering
- Rich media attachments
- Badge management
- Async/await API throughout
- Full test coverage
- Comprehensive documentation

### Platforms
- iOS 15.0+
- macOS 13.0+
- tvOS 15.0+
- watchOS 8.0+
